# Marketing ROI Analysis with OLS Regression

## Project Overview
This project analyzes the impact of advertising investments in **TV, Radio, and Social Media** on **Sales** using Python and statsmodels.  
The workflow includes:
- Exploratory Data Analysis (EDA) with descriptive statistics and visualizations
- Building and fitting an Ordinary Least Squares (OLS) regression model
- Validating model assumptions (linearity, normality, homoscedasticity)
- Interpreting R-squared, coefficients, and p-values in a business context
- Translating statistical results into actionable marketing ROI recommendations

The goal is to demonstrate how data science can transform raw marketing data into insights that guide budget allocation and maximize ROI.
